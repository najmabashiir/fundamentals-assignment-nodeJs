<!-- README -->

This Node.js script demonstrates basic array operations and a simple addition function:

1. It initializes an array `Myarry` with one object containing a name.
2. The `addobject` function adds a new object to the array and logs the updated array.
3. The `removeobject` function removes the last object from the array and logs the result.
4. The `add` function takes two numbers, adds them, and logs the sum.

To run the script, use:
    node app.js

You will see the array being updated and the sum of two numbers printed in the console.
